# HTML Practice

This repository contains a collection of simple HTML files created to practice fundamental HTML elements and concepts. Each file focuses on a specific topic, making it easy to understand and review how different tags work.

## 📁 File Structure & Topics

Here’s a list of the practice files and folders included:

- **Index/** – Starting practice files.
- **MetaTag/** – Explores metadata tags inside the `<head>` section (charset, description, viewport, etc.).
- **Anchor Tag/** – Demonstrates how to create hyperlinks using the `<a>` tag.
- **BlockAndInline/** – block‑level and inline elements practice.
- **Div and Span/** – Shows the usage of `<div>` (block container) and `<span>` (inline container).
- **EmphasisTag/** – **EmphasisTag/** – Practices text emphasis using various tags:
  - `<strong>` – bold text
  - `<em>` – italic/emphasized text
  - `<mark>` – highlighted text
  - `<del>` – deleted/strikethrough text
  - `<ins>` – inserted text
  - `<sub>` – subscript text
  - `<sup>` – superscript text
- **ImageTag/** – Covers embedding images with the `<img>` tag and its attributes.
- **LineBreak And Horizontal/** – Illustrates line breaks (`<br>`) and horizontal rules (`<hr>`) & HTML entities.
- **List/** – Examples of ordered (`<ol>`), unordered (`<ul>`), and definition lists (`<dl>`).
- **Semantic/** – Introduces semantic HTML elements like `<header>`, `<footer>`, `<article>`, `<section>`, and more.
